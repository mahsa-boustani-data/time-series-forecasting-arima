# Time Series Forecasting with ARIMA and TimesFM

## Objective
This project analyzes time series data and builds forecasting models using both classical statistical methods (ARIMA) and modern deep learning approaches (TimesFM).

## Methods
- ACF, PACF analysis
- Augmented Dickey-Fuller test
- ARIMA modeling
- Model selection using AIC and BIC
- Residual diagnostics
- Forecasting and evaluation (MSE)

## Results
- Best ARIMA model: ARIMA(2,1,1)
- ARIMA outperformed TimesFM in this case

## Tools
- R (forecast, tseries)
- Python (NumPy, Pandas, Matplotlib, PyTorch)

## Structure
- `R/arima_analysis.R`
- `python/timesfm_forecast.py`
- `report/report.pdf`
